Mori

A library for using ClojureScript's Persistent Data Structures and supporting API from the confort of vanilla JavaScript.

Rationale

JavaScript is a flexible and powerful dynamic programming language with a beautiful simple associative model. However this beautiful model comes along with ubiquitous mutability.

Modern JavaScript engines like V8, JavaScriptCore, and SpiderMonkey deliver the performance required to deliver efficient peristent data structures without incurring too high a cost.
